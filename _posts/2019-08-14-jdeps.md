There is the possibility of checking dependencies in maven using maven, I thought it would be interesting to do this with JRubYArt-2.0 build. Here is the relevant snippet of pom.rb.

```ruby
overrides do
  plugin :resources, '2.7'
  plugin :dependency, '2.8'
  plugin( :compiler, '3.8.0',
          'release' =>  '11' )
  plugin( :javadoc, '2.10.4',
          'detectOfflineLinks' =>  'false',
          'links' => [ '${processing.api}',
                       '${jruby.api}' ] )
  plugin( :jar, '3.1.0',
          'archive' => {
            'manifestEntries' => {
              'Class-Path' =>  'gluegen-rt.jar jog-all.jar'
            }
          } )
  plugin :jdeps, '3.0.0' do
    execute_goals 'jdkinternals', 'test-jdkinternals'
  end

end
```

To run jdeps from maven:-
```bash
mvn jdeps:jdkinternals
```
